Diagrama: `docs/diagrams/liquidacion_modalidad.svg`

Instrucciones para Confluence — cómo subir / incrustar este diagrama

1) Subir archivo (SVG) a Confluence
   - Abre la página en Confluence donde quieres insertar el diagrama.
   - Selecciona "Insert > Files and images" (o arrastra el archivo SVG al editor).
   - Una vez subido, haz click sobre la imagen y selecciona "Insertar" para que se muestre en la página.

2) Insertar como imagen embebida
   - Después de subir el archivo, con un click lo insertas como imagen en la página.
   - Ajusta tamaño (por ejemplo ancho 900px) para que se muestre correctamente.

3) Insertar como archivo descargable
   - En lugar de insertar, puedes simplemente arrastrar el archivo y seleccionar "Mostrar como enlace", útil si quieres que los usuarios lo descarguen.

4) Pegar el diagrama ASCII (opcional)
   - Si prefieres mantener la versión ASCII en bloque de código (monoespaciado), copia el contenido del archivo `docs/diagrams/liquidacion_modalidad_ascii.txt` (si existe) o el bloque que te pasé antes y pégalo dentro de un bloque de código en Confluence.
   - Usa el macro `{code}` (Clásico) o el bloque de código del editor nuevo. Selecciona "Plain text" o "None" para evitar coloreado.

5) Insertar el SVG en un panel
   - Para mejorar la legibilidad, puedes rodearlo con un panel (Insert > Panel) y dentro pegar la imagen o el bloque de código en fuente monoespaciada.

6) Recomendación: versión PNG
   - Para máxima compatibilidad con navegadores y posibles restricciones del plugin, considera exportar también una versión PNG y subirla junto con el SVG.
